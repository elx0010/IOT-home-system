# IoT Smart System (ESP32 + C++ + MQTT + Python + Node-RED)

A personal smart home system built with ESP32 nodes, an MQTT broker, a Python API backend, and Node-RED for orchestration and control.

## Features

- ESP32 connects over Wi-Fi
- Publishes temperature data via MQTT
- Listens for LED control commands
- Python FastAPI backend exposes REST endpoints
- Node-RED dashboard with live gauges and control

## Architecture

ESP32 → MQTT → Python API / Node-RED → Dashboard & Control

## Tech Stack

- ESP32
- Mosquitto MQTT
- Node-RED
- Python (FastAPI)
- GitHub
- C++
